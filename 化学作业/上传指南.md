# GitHub上传指南

## 化学作业文件已准备好：
1. `化学作业.html` - 网页版，可以直接浏览器打开
2. `化学作业整理.md` - Markdown文档版
3. `化学作业.tar.gz` - 压缩包版本

## 上传到 GitHub 的步骤：

### 方案1：使用 GitHub CLI（最简单）
```bash
# 安装 GitHub CLI
# 然后使用以下命令：
gh repo create 化学作业 --public
git add .
git commit -m "化学作业整理文档"
git push origin main
```

### 方案2：手动创建仓库
1. 登录 GitHub.com
2. 点击右上角 "+" → "New repository"
3. 输入仓库名：`化学作业`
4. 选择 "Public"
5. 点击 "Create repository"
6. 上传文件：
   - 点击 "Add file" → "Upload files"
   - 选择三个文件上传
   - 提交

### 方案3：使用 GitHub Desktop
1. 下载 GitHub Desktop
2. 登录 GitHub 账号
3. 创建新仓库
4. 拖拽文件到仓库
5. 提交并推送
